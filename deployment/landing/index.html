About the platform

alibalib is a workshop for full-stack applications and systems — from data-heavy business tooling to polished client-facing experiences. Each project is built, shipped, and run as a standalone service.

Everything here is built with .NET and Blazor, containerized with Docker, and deployed behind nginx on a single Linux host — with images published through a CI pipeline so each app updates independently.
