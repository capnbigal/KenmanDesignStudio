Corporate Campuses & Mixed-Use
------------------------------

Drop real project photographs into this folder to replace the generated SVG plates.
Recommended: landscape orientation, 1600x1200 or larger, .jpg or .webp.

To make a photo appear on a specific project, name it exactly like the generated
plate it should replace (e.g. va-24012-1.svg -> va-24012-1.jpg) — the app prefers a
real raster file over the .svg of the same name. Or re-seed after editing ProjectMedia.
